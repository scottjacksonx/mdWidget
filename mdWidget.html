Paragraphs

A paragraph is one or more lines of consecutive text, separated by one or more blank lines.

This means that two hard-wrapped lines without a blank line between them are considered to be part of the same paragraph.

To insert a <br/> break tag using Markdown, end a line with two or more spaces and press return.

Headers

Markdown has two types of headers -- Setext and atx-style.

Setext → This is an H1
============= 		This is an H2
-------------
atx-style → # This is an H1 ## This is an H2

Setext headers can have any number of ='s or -'s. You may "close" atx-style headers.

Blockquotes

Markdown uses email-style > characters for blockquoting.

> This is a blockquote
> with two paragraphs.
>
> The > character is
> on every line. 		> You can also be lazy and just put the > character on just the first line of the quote if the quote is one paragraph long.

Blockquotes can be nested.

Lists

Markdown supports ordered and un-ordered lists.

Ordered Unordered
1. Step One
2. Step Two
3. Step Three
 * Red
* Green
* Blue
 + Red
+ Green
+ Blue
 - Red
- Green
- Blue

If a list item has > 1 paragraph, paragraphs 2, 3,... must be indented by 4 spaces or a tab.

Code Blocks

Code blocks are for writing about programming or markup source code. To make a code block, indent every line of the block by >= 4 spaces or a tab. Within a code block, ampersands and angle brackets are automatically converted to HTML entities.

This is a normal paragraph.

    This is a code block.
Horizontal Rules

You can produce a horizontal rule tag (<hr />) by placing three or more hyphens, asterisks or underscores on a line by themselves. Spaces between hyphens or asterisks are fine.

* * *

***

- - - -
Links

Markdown supports two types of links -- inline and reference.

Inline This is [a link](http://example.com "Title here").
Reference This is [a link][id].
...
[id]: http://example.com "Title here"
Emphasis

Markdown treats asterisks (*) and underscores (_) as indicators of emphasis.

Markdown Output
*This* text _is_ italicised. This text is italicised.
**This** text __is__ bold. This text is bold.
Code

To indicate a span of code, wrap it with backtick quotes (`). Unlike a pre-formatted code block, a code span indicates code within a normal paragraph.

To print to stdout, use the `printf()` function.

Within a code span, ampersands and angle brackets are encoded as HTML entities automatically.

Images

Images are like links in that they are inserted in two styles: inline and reference.

Inline ![Alt text](path/to/img.jpg "Title here")
Reference ![Alt text][id]
...
[id]: path/to/img.jpg "Title here"

To specify dimensions, use the <img> HTML tag.
